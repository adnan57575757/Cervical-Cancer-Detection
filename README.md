## Features

- Multi-model approach using VGG16, ResNet50, and InceptionV3 pretrained on ImageNet.
- Two-phase fine-tuning strategy to optimize model performance.
- Ensemble learning (soft voting) to improve accuracy and robustness.
- Grad-CAM for explainable AI, visualizing important image regions for predictions.
- Detailed error and misclassification analysis to identify weak points.

## Dataset

- The dataset consists of cervical cell images categorized into five classes: Normal, Mild Dysplasia, Moderate Dysplasia, Severe Dysplasia, and Possible Malignancy.
- Data split: 70% training, 15% validation, 15% test.
- Data augmentation and class weights are applied to handle class imbalance.

## Model Architecture

- Backbone CNNs: VGG16, ResNet50, InceptionV3 (all pretrained on ImageNet).
- Custom classification head: Dense layers with dropout and L2 regularization.
- Explainability: Grad-CAM highlights image regions that influence model predictions.
- Ensemble: Soft voting aggregates predictions from all three models.

## Evaluation Metrics

- Accuracy
- ROC-AUC
- Cohen’s Kappa Score
- Per-class precision, recall, and F1-score
- Confidence distributions for each class

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/cervical-cancer-detection.git
   cd cervical-cancer-detection
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Place your image in the designated folder and run:
   ```bash
   python src/predict.py --image_path path/to/image.jpg --use_ensemble True
   ```

4. The script will output the predicted class, cancer stage, confidence, and risk level.

## Output Files

- `class_distribution.png`: Class distribution visualization.
- `sample_images.png`: Sample images per class.
- `confusion_matrix_*.png`: Confusion matrices for each model and ensemble.
- `roc_curves_*.png`: ROC curves per model.
- `final_model_comparison.png`: Comparison chart of all models.
- `perclass_radar_Ensemble.png`: Radar plot of per-class F1-scores (ensemble).
- `confidence_distribution.png`: Distribution of prediction confidence scores.
- `confidence_histogram.png`: Detailed confidence histogram plot.
